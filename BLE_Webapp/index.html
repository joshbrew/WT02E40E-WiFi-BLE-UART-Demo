Command reference

      

        All supported command shapes
        
    
          
  • status or wt status, prints full mode/BLE/Wi-Fi uptime status.
    • 
          
  • mode idle, disables BLE advertising and Wi-Fi request state.
    • 
          
  • mode ble, BLE only. This is the default boot mode.
    • 
          
  • mode wifi, Wi-Fi only.
    • 
          
  • mode both, BLE plus Wi-Fi.
    • 
          
  • ble on, ble off, ble status.
    • 
          
  • wifi on, wifi off, wifi status, wifi reconnect.
    • 
          
  • wifi cred list, lists stored runtime credentials.
    • 
          
  • wifi cred set <ssid> <password> [wpa2|auto|wpa3], clears runtime credentials and stores/selects one secured network.
    • 
          
  • wifi cred add <ssid> <password> [wpa2|auto|wpa3], adds another secured network.
    • 
          
  • wifi cred open <ssid>, stores/selects an open network.
    • 
          
  • wifi cred forget <ssid>, removes one stored runtime credential.
    • 
          
  • wifi cred clear, clears runtime credentials and falls back to compiled defaults.
    • 
          
  • tx ble <message>, sends a message over the BLE TX notify characteristic if subscribed.
    • 
          
  • tx uart <message>, writes a message to the firmware UART TX path.
    • 
          
  • tx wifi <ipv4> <port> <message>, sends a UDP packet over Wi-Fi.
    • 
          
  • tx both <ipv4> <port> <message>, sends using the combined TX path implemented in firmware.
    •
